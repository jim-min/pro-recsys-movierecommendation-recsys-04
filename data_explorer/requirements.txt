[기능]
1. 기본 기능은 train_ratings.csv로 user x item을 조회하는 것
    - user를 선택하면 해당 user가 점수를 매긴 시간순으로 점 분포를 보여주기
    - 초기 기간은 전체기간으로 하되 구간을 축소 확대할 수 있도록 함
    - user를 한명이 아니라 2명 이상 선택할 수도 있음
    - user가 두명 이상 선택할 때 점은 색깔과 모양(X, 다이아몬드, 별표 등)을 다르게 보일 것
    - 시간축은 하나로 겹치게 보일 수도 있고, user별로 축을 별도로 그려서 보이도록 선택할 수 있음
2. 처음에 데이터 저장 디렉토를 입력하도록 함
    - 현재 디렉토리(상대경로)를 보여주서 이동할 수 있도록 함
3. movie를 선택하면 item의 정보를 보여주기
    - 감독, 장르, 타이틀, 작가, 개봉년도 등 현재 data/train/ 폴더에 있는 파일을 이용하여 얻을 수 있는 정보
4. 통계 정보
    - user별 rating한 건수, 일별, 주별, 월별, 년도별 선택
    - user 인터렉션 건수를 가장 많은 user를 보여주기 ( top k, k는 선택할 수 있도록 함)
    - user 인터렉션 건수를 가장 적은 user를 보여주기 ( bottom k, k는 선택할 수 있도록 함)
    - user 인터렉션 건수를 가장 많은 item을 보여주기 ( top k, k는 선택할 수 있도록 함)
    - user 인터렉션 건수를 가장 적은 item을 보여주기 ( bottom k, k는 선택할 수 있도록 함)
    - user 인터렉션 건수를 가장 많은 user와 item을 보여주기 ( top k, k는 선택할 수 있도록 함)
    - user 인터렉션 건수를 가장 적은 user와 item을 보여주기 ( bottom k, k는 선택할 수 있도록 함)
5. 화면 구성
    - 왼쪽 슬라이드: 데이터 디렉토리 선택, user 선택(다중 선택 가능, from - to 로도 가능), 시간축 구간은 slider로, 시간축 별도 여부 선택
    - 오른쪽 슬라이드: user interactions over time(기본), user temporal stats, item 인포 조회, 통계 정보
